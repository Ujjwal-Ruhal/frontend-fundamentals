My HTML Basics Notes

Yeh mere HTML ke notes hain jisme main basic tags aur unka use likh raha hoon.

1. Headings

Headings titles ke liye use hoti hain. h1 sabse bada aur h6 sabse chhota hota hai.

This is Heading 1


This is Heading 2


This is Heading 3

Output:

This is Heading 1

This is Heading 2

This is Heading 3

2. Paragraph

Paragraph normal text ke liye use hota hai.

This is a paragraph.

Output:

This is a paragraph.

3. Line Break

br tag new line dene ke liye use hota hai.

This is line 1 
 This is line 2

Output:

This is line 1
This is line 2
